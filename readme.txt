Search And Destroy

- Originalmente desenvolvido em java, resolvi tentar migrar para uma implementação em javascript html5, A Engine do jogo ficou praticamente identica a original, o algoritmo de pathfinder não está 100% preciso, e ainda o gameplay precisa ser aprimorado.


ROADMAP

- Interface interna do jogo (GUI)
- Telas de abertura/Opções de jogo
- Instruções de jogo, hotkeys
- Aprimoramento do pathfinder
- Criação de outros mapas
- HomePage com descrição do projeto Search and Destroy (Chrome Experiment)



FUTURO

- Novos modos de jogos, tanto single player, como multi-player
- Editor de mapas
